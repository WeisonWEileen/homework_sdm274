# how to run

### run scripts
configure options in ```conf/config.yaml```  
and run
```
cd hw2
python3 -m scripts.gd_reg
```
### 维度说明
代码中的某些数据维度说明
$$
\begin{aligned} &X \in \mathbb{R}^{m \times (n + 1)} \ &W \in \mathbb{R}^{(n + 1) \times 1} \ &\text{grad} \in \mathbb{R}^{(n + 1) \times 1} \end{aligned} 
$$
